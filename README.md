# Prodigy InfoTech Internship Task Programs

This workspace contains a set of Python task programs created as part of an internship assignment. Each project folder contains a small utility or game that demonstrates basic Python programming, algorithms, and file handling skills.

## Project Overview

### `PRODIGY_SD_01`
- **Program:** `TemperatureConversionProgram.py`
- **Description:** Temperature conversion tool that converts between Celsius, Fahrenheit, and Kelvin.
- **Notes:** Includes interactive unit selection and conversion logic.

### `PRODIGY_SD_02`
- **Program:** `Guessing_Game.py`
- **Description:** Number guessing game where the player tries to guess a randomly selected number.
- **Notes:** Provides feedback on whether guesses are too high or too low and counts attempts.

### `PRODIGY_SD_03`
- **Program:** `ContactManagementSystem.py`
- **Description:** Simple contact manager with add, edit, delete, and list functionality.
- **Notes:** Uses JSON file storage for persistence.

### `PRODIGY_SD_04`
- **Program:** `SudokuSolver.py`
- **Description:** Sudoku solving program that accepts an unsolved puzzle and completes it using backtracking.
- **Notes:** Validates rows, columns, and 3x3 subgrids.

### `PRODIGY_SD_05`
- **Program:** `WebScraping.py`
- **Description:** Web scraping utility that extracts product information from a practice e-commerce website and saves the data to a CSV file.
- **Notes:** Uses Python standard libraries for HTTP requests, HTML parsing, and CSV output.

## How to Use

1. Open the relevant project folder.
2. Run the Python script for that task.
3. Consult the folder README or script comments for specific usage details.

## Internship Task Summary

These programs were developed to demonstrate:

- Basic Python scripting
- User input and interaction
- Data parsing and validation
- File I/O and persistence
- Algorithm implementation
- Web scraping and structured output generation

## Notes

Each project is self-contained, and the code is written for readability and learning rather than production deployment. Feel free to explore each folder and run the scripts individually.
